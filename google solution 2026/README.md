# TruthMark Prototype

A static website prototype for TruthMark, an AI-powered content authentication concept.

## Overview

TruthMark combines invisible watermarking, blockchain-backed authenticity certificates, and AI verification to protect creators and help users verify digital content.

## Files

- `index.html` — landing page layout and content.
- `styles.css` — responsive visual styling.
- `script.js` — smooth scroll behavior for internal navigation.

## Run locally

Open `index.html` in your browser, or use your editor's Live Server extension for a local preview.
